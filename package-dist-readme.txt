mycmux セットアップガイド
========================

mycmux は Windows 向けの開発用ターミナルワークスペースです。
複数ペイン分割・セッション管理・Claude Code との親和性を重視して作られています。

1. 動作環境
-----------
- Windows 10 / 11 (x64)
- WebView2 ランタイム（インストーラが自動で導入します）
- 管理者権限は不要

2. インストール手順
-------------------
(1) このフォルダ内の mycmux-setup.exe をダブルクリック
(2) Windows SmartScreen の警告が出た場合は「詳細情報」→「実行」
    ※ 社内ビルドで署名無しのため表示されます。問題ありません
(3) セットアップウィザードの指示に従う（デフォルトで OK）
(4) 完了後、スタートメニューまたはデスクトップショートカットから起動

3. 初回起動
-----------
起動するとシェルが自動検出されて 1 ペイン開きます。
Windows では Git Bash を優先的に検出します。

4. キーボードショートカット
---------------------------
基本操作:
  Ctrl+Shift+Enter      ペイン拡大 / 元に戻す
  Ctrl+Shift+→ / ↓       ペインを右 / 下に分割
  Ctrl+Shift+W          ペインを閉じる
  Ctrl+Shift+T          新しいタブを開く
  Ctrl+Shift+N          通知一覧パネルを開く
  Ctrl+F                 ターミナル内検索
  Ctrl+,                 テーマ切替
  Ctrl+Shift+P          コマンドパレット

テキスト操作:
  Ctrl+V                 貼り付け
  Ctrl+C                 選択範囲コピー（選択中のみ）、未選択なら SIGINT
  Shift+Enter            改行挿入（Claude Code 等の複数行入力）

5. ペイン分割
-------------
ペインタブバー右側のアイコンから:
  ├─ 右分割 (split right)
  ├─ 下分割 (split down)
  ├─ 拡大 / 元に戻す (zoom)
  └─ ペインを閉じる

分割後、ペイン境界をドラッグしてサイズ調整できます。

6. テーマ切替
-------------
Ctrl+, でテーマスイッチャーを開く。17 テーマ（dark / light）を収録:
  夜カフェ / 夜霧 / 極夜 / 黒曜 / 夜景 / 篝火 / 宵藍 / 墨色 /
  真夜中 / 深海 / 生成り / 朝凪 / 桜 / 若葉 / 藤 / 青空 / 夕焼け

7. 通知について
----------------
非アクティブペインで Claude Code 等が承認を要求すると、ペインの
サイドバーに amber のバッジと音で通知されます。作業完了時には
emerald の緑バッジが出ます。

8. トラブルシューティング
-------------------------
Q. ダブルクリックしても起動しない
A. WebView2 ランタイムが入っていない可能性があります。インストーラが
   自動で入れますが、既存環境と競合している場合は Microsoft の公式
   ページから「Microsoft Edge WebView2 Runtime」を手動インストール
   してください。

Q. Windows Defender SmartScreen に止められる
A. 社内ビルドで証明書署名をしていないため警告が出ます。「詳細情報」
   →「実行」で続行してください。

Q. コピーができない
A. xterm の選択範囲は自動的にクリップボードにコピーされます。貼り付
   けは Ctrl+V です。右クリックで単語選択もできます。

Q. フォント / ターミナル色がおかしい
A. Ctrl+, でテーマを切り替えてみてください。現在のテーマが好みに合
   わない場合はここから変更できます。

Q. Claude Code の承認通知が聞こえない
A. Web Audio API を使っています。ブラウザ（WebView2）の音声が
   ミュートされていないか確認してください。

9. アップデート
---------------
新しい mycmux-setup.exe を受け取ったら、同じ手順で上書きインストー
ルできます。設定・テーマ・キーバインドは引き継がれます。

10. 不具合の報告
----------------
社内チャンネル（または宮崎まで）にご連絡ください。
画面のスクリーンショットや再現手順があると助かります。
