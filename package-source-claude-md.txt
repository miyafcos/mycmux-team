# mycmux — source distribution

これはターミナルワークスペース mycmux (Tauri v2 + React) のソース配布版です。
配布バージョン: v{VERSION} ({DATE}) / 作者: Miyazaki Efu / ライセンス: GPL-3.0

このフォルダは「配布時点で凍結されたスナップショット」です。日々のアップデートは
反映されません。新しい版が必要なときは、作者から新しい zip を受け取ってください。

## Claude Code へのお願い

ユーザーから「セットアップして」「インストールして」「install」等の依頼が来たら、
同じフォルダにある `INSTALL.md` を読み、そこに書かれた手順を上から順に実行してください。
各ステップの前に実行コマンドを1行で告知し、失敗したらすぐ止めてユーザーに相談する方針です。

## プロジェクト要点

- フロントエンド: `src/` (React 19 + TypeScript + Zustand + xterm.js)
- バックエンド: `src-tauri/src/` (Rust + Tauri v2 + portable-pty)
- ビルド成果物: `src-tauri/target/release/mycmux.exe`
- 設定保存先: `%APPDATA%/com.miyazaki.mycmux/data.json`
- identifier (`com.miyazaki.mycmux`) は変更しないでください。WebView2 のデータ領域と紐付いています。

## やらないこと

- git commit / push しない(リポジトリ履歴は配布対象外)
- package.json の依存を勝手にメジャーアップデートしない
- `src-tauri/tauri.conf.json` の identifier, productName を変更しない
- ソース配布なので `.claude/` などの作者環境は同梱されていません。ユーザーの環境を尊重してください。
