mycmux v{VERSION} — ソース配布版 ({DATE})
==============================================

mycmux は Windows 向けのターミナルワークスペースです。
ペイン分割・セッション管理・Claude Code との相性を重視して作られています。
Tauri v2 (Rust) + React 19 + xterm.js、GPL-3.0。

この zip は「作者環境に依存しないソース一式」です。お手元で自分でビルドして
使う形になります。バイナリ配布ではありません。


===================================
一番かんたんな入れ方: Claude Code に任せる
===================================

1. この zip をどこかに展開する (例: C:\Users\<あなた>\mycmux-src\)
2. そのフォルダで Claude Code を起動する (VS Code 拡張でも CLI でも可)
3. 最初のメッセージで「このフォルダの mycmux をセットアップして」と伝える

Claude Code がフォルダ内の `CLAUDE.md` と `INSTALL.md` を読み、前提ツールの
チェックからビルド、ショートカット作成まで順に進めてくれます。
途中で「Node.js が入っていない」等と言われたら案内に従ってください。


===================================
手動で入れたいとき
===================================

前提 (いずれも無料):
  - Windows 10 / 11 (x64)
  - Node.js 20 以上        https://nodejs.org/ja/download
  - Rust (rustup)           https://rustup.rs/
  - MSVC Build Tools 2022   https://visualstudio.microsoft.com/visual-cpp-build-tools/
    ※ インストーラで「C++ によるデスクトップ開発」ワークロードを選ぶこと

手順:
  1. 展開したフォルダで PowerShell を開く
  2. npm install                                        (数分)
  3. powershell -ExecutionPolicy Bypass -File .\build-full.ps1   (初回 5〜15 分)
  4. src-tauri\target\release\mycmux.exe をお好きな場所にコピー
  5. ダブルクリックで起動

詳細は `INSTALL.md` を参照してください (Claude Code 用の手順書ですが、
人間が読んでもそのまま使えます)。


===================================
主なキーボードショートカット
===================================

  Ctrl+Shift+Enter     ペイン拡大 / 戻す
  Ctrl+Shift+→ / ↓      右 / 下に分割
  Ctrl+Shift+W         ペインを閉じる
  Ctrl+Shift+T         新しいタブ
  Ctrl+Shift+N         通知パネル
  Ctrl+F               ターミナル内検索
  Ctrl+,               テーマ切替 (17 種)
  Ctrl+Shift+P         コマンドパレット
  Shift+Enter          改行挿入 (Claude Code 等の複数行入力用)


===================================
既知の注意点
===================================

- 初回起動時に Windows SmartScreen が警告を出します (署名無しビルドのため)。
  「詳細情報」→「実行」で進んでください。
- 起動中の mycmux.exe を上書きビルドするとエラーになります。
  ビルド前に mycmux を終了してください。
- ブラウザペイン機能 (ghostty-web) は Windows ではスタブです。未対応。
- 設定・テーマは %APPDATA%\com.miyazaki.mycmux\data.json に保存されます。
